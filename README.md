# Snake_spelling_Game

### 介紹

本作品結合「貪吃蛇」與「英文拼字學習」製作而成，  
透過遊戲方式提升單字記憶與反應能力。

本作品基於 pygame 貪吃蛇基礎架構進行延伸開發，  
並自行設計拼字系統、雙選項機制與遊戲介面。

--------------------

### 遊戲特色
- 英文單字拼寫系統（逐字完成）
- 結合遊戲與學習的互動體驗
- 正確 / 錯誤字母雙選項設計

--------------------

### 操作方式

- 方向鍵控制蛇移動
- 吃正確字母完成單字拼字

--------------------

### 遊戲說明

- 右上角顯示目前分數
- 左上角顯示目前單字中文
- 螢幕同時顯示一正確一錯誤字母
- 吃到正確字母：加分並推進拼字進度
- 吃到錯誤字母：扣分
- 撞牆或撞到自己：結束遊戲
