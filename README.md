# Snake_spelling_Game

操作方式：

* 方向鍵控制蛇移動
* 吃正確字母完成單字拼字

遊戲說明：

* 右上角顯示目前分數
* 左上角顯示目前單字中文
* 螢幕同時顯示一正確一錯誤字母
* 每吃一個正確字母會逐步完成拼字並加分
* 每吃一個錯誤即扣分
* 撞牆或撞到自己會結束遊戲
